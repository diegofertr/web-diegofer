## Let syntax

With `Let` we can create variables that can change along the code

```javascript
let nombre = 'Pedro'
console.log(nombre); // Pedro

nombre = 'Fernando'
console.log(nombre) // Fernando
```

## Const syntax

With _Const_ we can create variables that can't change along the code, this is a constant variable

```javascript
const precio = 23;
console.log(precio) // 23

precio = 456; // Uncaught TypeError: Assignment to constant variable.
```
